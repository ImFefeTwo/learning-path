# How Contribute? 

Thanks for you interest in these notes!

---

## Report Issues

If you find an error:  

- Open an issue explaining the error
- Point out the file and line number
- If you now how to solve the issue, even better!

## Suggest Improvements

- Ideas to better explain concepts
- Better code examples
- Additional resources

## Process 
1. Fork the repository
2. Create a branch with your changes
3. Commit your changes
4. Open a Pull Request

> **Note:** Again, thanks for considering been part of this proyect!
> Remember, this is a learning journey, so it's not aiming for perfection.
